电脑常用快捷键

快捷键 功能
Ctrl + C 复制选中的内容
Ctrl + V 粘贴复制的内容
Ctrl + Z 撤销上一步操作
Ctrl + A 全选当前页面内容
Alt + Tab 在打开的应用程序之间切换
Win + D 显示桌面
Win + E 打开文件资源管理器
Win + L 锁定计算机
Ctrl + Shift + Esc 打开任务管理器
Win + R 打开“运行”对话框
Win + I 打开设置
Win + S 打开搜索
Win + A 打开操作中心
Win + X 打开快捷菜单
Win + P 投影屏幕
Win + , 临时查看桌面
Win + Tab 打开任务视图
Win + Shift + S 截取屏幕截图
Win + Ctrl + D 创建新的虚拟桌面
Win + Ctrl + F4 关闭当前虚拟桌面
Win + Ctrl + ←/→ 切换虚拟桌面
Alt + F4 关闭当前窗口
Ctrl + F 查找
Ctrl + S 保存
Ctrl + N 新建窗口
Ctrl + O 打开文件
Ctrl + P 打印
Ctrl + W 关闭当前标签页或窗口
Ctrl + Tab 切换标签页
Ctrl + Shift + N 新建文件夹
Ctrl + Shift + T 重新打开最近关闭的标签页
Ctrl + Shift + Esc 打开任务管理器
Ctrl + Alt + Del 打开安全选项
Shift + Delete 永久删除文件
F2 重命名文件
F3 查找文件
F5 刷新
F11 全屏显示
